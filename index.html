Passkey Wallet

Ethereum smart contract wallet powered by native passkeys. No seed phrases. No private key export. Biometric-only.

Secure Enclave Keys P-256 keys generated and stored in Apple's Secure Enclave. Private keys never leave the device.
Deterministic Wallets Smart contract wallet address derived via CREATE2 from your passkey's public key.
On-Chain Verification Signatures verifiable on-chain via the RIP-7212 P-256 precompile.
